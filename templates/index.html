How It Works

This app runs entirely locally — no data is sent to external AI services. It uses Retrieval-Augmented Generation (RAG) to answer questions from real NASA content.

Web Scraping 619 mission pages from NASA's A-to-Z index
Chunk & Embed Text split into ~4,000 chunks, embedded with nomic-embed-text
Vector Store ChromaDB with cosine similarity search
Your Question Matched to the most relevant chunks via hybrid search
LLM Response Generated locally by llama3.2:3b via Ollama
Performance note: This app uses llama3.2:3b, a small 3-billion parameter model running locally via Ollama. Responses may be less accurate or detailed compared to larger commercial models. The tradeoff is full privacy — your questions never leave this server.
